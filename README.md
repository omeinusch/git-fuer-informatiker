Git für Informatiker

Die aktuellste Version der Präsentation kann unter http://omeinusch.github.io/git-fuer-informatiker betrachtet werden.

Termine:
- 7. Mai 2013
- 28. Mai 2013

von Oliver Meinusch, oliver.meinusch@stud.h-da.de
